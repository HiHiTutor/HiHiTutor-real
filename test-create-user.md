## 測試步驟

### 測試1: 創建中小學教育tutor
1. 選擇課程分類：中小學教育
2. 子科目：選擇"不限教育階段"或特定階段
3. 可教授科目：選擇多個科目（可跨小學和中學）
4. 教學模式：選擇"皆可"
5. 教學子模式：不選擇（應該允許）
6. 填寫其他必要信息
7. 點擊創建

### 測試2: 創建面授模式tutor
1. 選擇課程分類：任意
2. 教學模式：選擇"面授"
3. 教學子模式：必須選擇（一對一/小班教學/補習社）
4. 填寫其他必要信息
5. 點擊創建

### 測試3: 編輯tutor科目
1. 進入已創建的tutor詳情頁面
2. 點擊"編輯用戶"
3. 切換到"導師資料"標籤頁
4. 修改"可教授科目"
5. 點擊"儲存"

### 測試4: 驗證錯誤處理
1. 嘗試創建tutor但不選擇科目
2. 嘗試創建tutor但堂費低於100元
3. 嘗試創建面授模式tutor但不選擇教學子模式
